# Esther's Portfolio

This is a personal portfolio website built using React, React Router, and Material UI. It is fully responsive and mobile-friendly.

## Getting Started

To get started, clone the repository and install the dependencies:

```
git clone https://github.com/Hestia-Arc/Portfolio-vercel.git
cd Portfolio-vercel
npm install
```

Once the dependencies are installed, you can start the development server:

```
npm start
```

The development server will run on port 3000. You can open the website in your browser by visiting http://localhost:3000.

## Code Structure

The codebase is structured as follows:

* `package.json`: The package.json file contains the project's dependencies and scripts.
* `public/index.html`: The index.html file is the entry point of the website. It includes the HTML code for the website's header, body, and footer.
* `src/App.js`: The App.js file is the main component of the website. It contains the routes for the different pages of the website.
* `src/components/home/Home.js`: The Home.js file is the component for the home page of the website.
* `src/components/about/About.js`: The About.js file is the component for the about page of the website.
* `src/components/contact/Contact.js`: The Contact.js file is the component for the contact page of the website.
* `src/components/project/Project.js`: The Project.js file is the component for the project page of the website.
* `src/theme.js`: The theme.js file contains the theme for the website.
* `src/images/back.jpg`: The back.jpg file is the background image for the website.

## Running the Website

To run the website, you can use the following command:

```
npm start
```

The development server will run on port 3000. You can open the website in your browser by visiting http://localhost:3000.
